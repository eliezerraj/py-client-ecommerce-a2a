## py-client-ecommerce-a2a

A multi-agent using aws strands

## create venv

    python3 -m venv .venv

## activate

    source ../.venv/bin/activate

## install requirements

    python3 ./a2a-agent/main.py

## install requirements

    pip install -r requirements.txt
