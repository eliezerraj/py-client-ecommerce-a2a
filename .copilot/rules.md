# Copilot Rules

These rules guide AI-generated code.

## General Principles

- Prefer simple, readable implementations.
- Avoid premature optimization.
- Write modular code.
- Follow SOLID principles.

## File Size

- Max 300 lines per file.
- Max 50 lines per function.

## Naming

- Functions: verbNoun
- Classes: PascalCase
- Variables: snake_case

Example:

UserService
createUser()
getUserById()
source_agent
